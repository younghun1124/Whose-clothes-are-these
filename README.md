# Whose-clothes-are-these  

가장 비슷한 옷 사진을 찾아주는 소프트웨어입니다.  

## 문제상황  

### "이거 누구 옷이야?"  

자식들이 다 크고나니 엄마와 딸, 아빠와 아들, 딸과 아들의 옷이 헷갈릴 때가 있습니다.  
빨래를 갤 때마다 집에 가족이 모두 있는 것이 아니기 때문에 물어보기도 곤란합니다.  

### "내 옷은 다 어디갔지?"  

자꾸 섞이는 옷들 때문에 내 옷이 동생 방의 옷장에 수개월간 잠들어 있는 경우가 있습니다.
